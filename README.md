# Prisma + tRPC

## Setup

**yarn:**
```bash
yarn
yarn dx
```

### Requirements

- Node >= 14

## Development

### Commands

```bash
yarn build      # runs `prisma generate` + `prisma migrate` + `next build`
yarn db-nuke    # resets local db
yarn dev        # starts next.js
yarn dx         # runs migrations + seeds + starts next.js 
yarn test:unit  # runs normal jest unit tests
```
